# Express-JWT-Auth

-   Express typescript
-   JWT auth
-   Knex as Querybuilder
-   Mysql as database
-   SWC as fast compiler

## install

```bash
git clone https://github.com/amdkholil/Express-JWT-Auth.git myapp

cd myapp

npm install
```

## run development

```bash
npm run dev
```

## build for production

```bash
npm run build
```
